# Student Retention & Academic Performance Analytics

An end-to-end Power BI analytics project simulating a real Registrar's Office / Institutional Research use case: understanding why students drop out, quantifying revenue at risk, and flagging at-risk students for intervention.

Built following a real industry analyst workflow:
Business Questions → Break Down → Hypothesis → Data → Analysis → Insight → Communicate → Decision
